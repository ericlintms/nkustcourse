# 投影片 4/4: 勝率與賺賠比

**迷思：勝率越高越好？**

不一定！一個勝率 90% 的策略，也可能是虧錢的。

---

## 關鍵在於「期望值」

`期望值 = (勝率 * 平均獲利) - (賠率 * 平均虧損)`

一個能賺錢的策略，期望值必須 > 0。

---

## 賺賠比 (Reward-to-Risk Ratio, RRR)

* **定義**: `平均獲利 / 平均虧損`
* 它告訴你，你賺錢的時候平均賺多少，是你賠錢時平均賠的幾倍。

## 策略類型

1. **高勝率，低賺賠比**
    * 每次都賺一點點，但偶爾一次大賠就把獲利全吐回去。
    * 例子: 選擇權賣方策略。
    * `勝率 90%，賺賠比 0.1` => 期望值 = (0.9 *1) - (0.1* 10) = -0.1 (虧錢)

2. **低勝率，高賺賠比**
    * 大部分交易都小賠，但靠少數幾次大賺來獲利。
    * 例子: 趨勢追蹤策略。
    * `勝率 30%，賺賠比 3` => 期望值 = (0.3 *3) - (0.7* 1) = 0.2 (賺錢)

**結論: 勝率和賺賠比必須一起看，才能判斷策略的真實獲利能力。**
