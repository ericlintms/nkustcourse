# 投影片 1/4: 別被總報酬率騙了

**問題：策略A一年賺了 50%，策略B一年賺了 50%，它們一樣好嗎？**

不一定！看看它們的資金曲線：

* **策略 A**: 資金曲線平滑向上，最大只回檔了 5%。
* **策略 B**: 資金曲線像雲霄飛車，中間一度虧損了 40% 才漲回來。

雖然最終報酬一樣，但策略 A 的過程讓人安心，策略 B 的過程則讓人夜不能寐，甚至可能在虧損 40% 時就因為受不了而手動平倉，根本撐不到最後的 50% 獲利。

---

**結論：只看報酬，不看風險，是業餘投資者的通病。**

一個好的策略，追求的是 **「風險調整後的高報酬」**，也就是我們常說的「高 CP 值」。
