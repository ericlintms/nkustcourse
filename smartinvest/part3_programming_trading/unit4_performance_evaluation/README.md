# 單元 4: 回測與績效評估

回測報告產出了一大堆數字，哪些最重要？如何客觀地評估一個策略的好壞？本單元將深入探討關鍵的績效指標。

## 學習目標

- 學會解讀 `backtesting.py` 的回測報告
- 理解並能解釋夏普比率 (Sharpe Ratio) 的意義
- 理解並能解釋最大回撤 (Max Drawdown) 的重要性
- 了解勝率與賺賠比的關係

### 內容

1. **解讀回測報告**
    - `backtesting.py` 報告中的各項指標概覽。
2. **衡量報酬：年化報酬率**
    - `Return (Ann.) [%]`
3. **衡量風險：最大回撤**
    - `Max. Drawdown [%]`：策略可能面臨的最大資金回檔，風險承受度的關鍵指標。
4. **風險調整後報酬：夏普比率**
    - `Sharpe Ratio`：承受每單位風險，可以獲得多少超額報酬？衡量策略「效率」的指標。
5. **交易品質：勝率與賺賠比**
    - `Win Rate [%]`
    - `Avg. Trade [%]` vs. `Best Trade [%]` vs. `Worst Trade [%]`
    - 高勝率不一定賺錢，需要搭配賺賠比來看。
