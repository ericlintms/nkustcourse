# 單元 1: 程式交易概論

本單元將介紹程式交易的基本概念，以及一個典型交易策略的生命週期。

## 學習目標

- 理解什麼是程式交易 (Algo-Trading)
- 了解程式交易的優點與風險
- 認識一個交易策略從發想到上線的完整流程

### 內容

1. **什麼是程式交易？**
    - 定義：將交易邏輯與規則寫成程式，讓電腦自動執行。
    - 與手動交易的比較。
2. **優點與風險**
    - 優點：紀律性、速度、處理大量資料的能力。
    - 風險：程式錯誤 (Bug)、過度最佳化 (Overfitting)、市場結構改變。
3. **交易策略的生命週期**
    - 策略發想 (Idea Generation)
    - 資料準備 (Data Preparation)
    - 策略回測 (Backtesting)
    - 績效評估 (Performance Evaluation)
    - 模擬交易 (Paper Trading)
    - 實盤上線 (Live Trading)
