# 投影片 3/4: 交易策略的生命週期 (1)

一個嚴謹的量化交易策略，從無到有，通常會經過以下階段：

## 1. 策略發想 (Idea Generation)

* 一切的起點。這個想法可以來自於：
  * 學術論文 (e.g., Fama-French 三因子模型)
  * 經典技術指標 (e.g., MA, RSI)
  * 市場觀察 (e.g., "每當A事件發生，B股票似乎都會漲")
  * 基本面邏輯 (e.g., 尋找營收連續成長的公司)
* **重點**: 將想法轉化為 **明確、可量化** 的規則。

## 2. 資料準備 (Data Preparation)

* 根據你的策略，收集所需要的歷史資料。
* 常見資料: 開高低收成交量 (OHLCV)、財報數據、法人買賣超...
* **重點**: 資料的 **品質** (準確性、完整性) 會直接影響回測結果的可信度。垃圾進，垃圾出 (Garbage in, garbage out)。

## 3. 策略回測 (Backtesting)

* **核心步驟**。將你的交易規則應用在歷史資料上，模擬過去的交易情況。
* **目的**: 看看這個策略在過去的表現如何？是否賺錢？風險多大？
