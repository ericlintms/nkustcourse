# 投影片 2/4: `backtesting.py` 介紹

## 一個輕量、快速、易用的回測框架

---

## 主要特色

1. **簡單易用**: API 設計直觀，幾行程式碼就能完成一個回測。
2. **向量化回測**: 底層使用 NumPy 進行計算，速度非常快。
3. **與 Pandas 完美整合**: 資料格式就是我們熟悉的 DataFrame。
4. **內建常用指標**: 內建了 SMA, EMA, RSI 等常用技術指標。
5. **策略最佳化**: 可以自動幫你尋找策略的最佳參數組合。
6. **精美報表**: 自動產生包含各種績效指標和圖表的 HTML 報表。

---

## 安裝

```bash
pip install backtesting
```

## 核心概念

* `Backtest`: 主要的回測物件，負責管理整個回測流程。
* `Strategy`: 你定義交易邏輯的地方。
