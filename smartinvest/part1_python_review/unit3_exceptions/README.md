# 單元 3: 錯誤處理與例外

在撰寫程式時，預期並處理可能發生的錯誤是非常重要的，這能讓你的程式更穩定、更可靠。

## 學習目標

- 了解 Python 中的錯誤 (Error) 和例外 (Exception)
- 學會使用 `try...except` 來捕捉和處理例外
- 學習如何自訂例外類別

## 內容

1. **基本例外處理**:
    - `try...except` 語法
    - 處理特定類型的例外
    - `else` 和 `finally` 子句
2. **引發例外**:
    - 使用 `raise` 主動引發例外
3. **自訂例外**:
    - 為什麼需要自訂例外？
    - 如何建立自己的例外類別
