# 單元 1: Python 核心語法

本單元將快速複習 Python 的核心語法，為後續的課程打下穩固的基礎。

## 學習目標

- 熟悉 Python 的基本資料型態與資料結構
- 掌握流程控制的語法
- 了解如何定義與使用函式及模組

## 內容

1. **資料型態與資料結構**:
    - 數字 (Integer, Float)
    - 字串 (String)
    - 列表 (List)
    - 字典 (Dictionary)
2. **流程控制**:
    - `if`, `elif`, `else`
    - `for` 迴圈
    - `while` 迴圈
3. **函式與模組**:
    - 如何定義自己的函式
    - 如何引用 Python 內建或第三方模組
