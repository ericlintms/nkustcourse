# 單元 4: 選股策略

市場上有上千支股票，我們該如何從中挑選出值得投資的標的？本單元將介紹幾種經典的選股策略，並用程式實作。

## 學習目標

- 了解不同選股策略背後的邏輯
- 學會如何利用 Pandas 進行多條件篩選，實作選股
- 實作基於價值面和技術面的選股範例

## 內容

1. **價值投資策略 (Value Investing)**:
    - 尋找價格被市場低估的股票。
    - 指標：本益比 (P/E Ratio)、股價淨值比 (P/B Ratio)、股息殖利率 (Dividend Yield)。
2. **成長股投資策略 (Growth Investing)**:
    - 尋找營收和利潤高速成長的公司。
    - 指標：營收成長率 (Revenue Growth Rate)、每股盈餘成長率 (EPS Growth Rate)。
3. **技術指標選股**:
    - 利用技術指標來尋找買入訊號。
    - 指標：RSI (相對強弱指標) 進入超賣區、MACD 黃金交叉。
