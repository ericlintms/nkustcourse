# 單元 1: 智慧投資基礎

在開始用程式進行投資分析之前，我們需要對金融市場和主要的分析方法有基本的認識。

## 學習目標

- 了解金融市場的基本組成
- 認識三大分析法：基本面、技術面、籌碼面

## 內容

1. **金融市場概論**:
    - 股票、債券、基金、期貨... 我們在哪裡交易？
    - 初級市場 vs. 次級市場
2. **三大分析法**:
    - **基本面分析**: 關注公司的「內在價值」。(e.g., 財報、產業前景)
    - **技術面分析**: 關注「價格與成交量」的歷史圖表，相信歷史會重演。
    - **籌碼面分析**: 關注「誰」在買賣股票。(e.g., 三大法人、主力)
