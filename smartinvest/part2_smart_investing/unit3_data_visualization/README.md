# 單元 3: 數據可視化 - Matplotlib

一張圖勝過千言萬語。數據可視化是探索資料、驗證策略、呈現結果的關鍵技能。

## 學習目標

- 學會使用 Matplotlib 繪製基本的圖表
- 能夠客製化圖表的標題、標籤和樣式
- 了解如何繪製金融領域常用的 K線圖

## 內容

1. **Matplotlib 基礎**:
    - `pyplot` 模組介紹
    - 繪製折線圖 (Line Chart)
    - 繪製長條圖 (Bar Chart)
2. **圖表客製化**:
    - 加入標題和 X/Y 軸標籤
    - 調整線條顏色、樣式
    - 顯示圖例 (Legend)
3. **繪製 K線圖**:
    - 使用 `mplfinance` 套件 (專為金融圖表設計)
    - 繪製包含成交量的 K線圖
