# 單元 2: 數據處理與分析 - Pandas & NumPy

Pandas 和 NumPy 是 Python 進行數據科學和金融分析的兩大基石。

## 學習目標

- 掌握 Pandas 的核心資料結構：DataFrame 和 Series
- 學會使用 Pandas 進行資料的讀取、選取、過濾和清理
- 了解 NumPy 的基本陣列操作

## 內容

1. **NumPy 基礎**:
    - 建立 NumPy 陣列 (Array)
    - 陣列的數學運算
2. **Pandas Series**:
    - 帶有索引的一維陣列
3. **Pandas DataFrame**:
    - 像是 Excel 表格的二維資料結構
    - 從 CSV 讀取資料到 DataFrame
    - 選取欄 (Column) 和列 (Row)
    - 條件過濾
    - 處理遺失值 (Missing Data)
