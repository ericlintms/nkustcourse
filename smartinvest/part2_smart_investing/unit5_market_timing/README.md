# 單元 5: 擇時策略

選好了股票，下一步的難題是：「什麼時候買？什麼時候賣？」本單元將介紹基於技術指標的擇時策略。

## 學習目標

- 了解擇時策略的重要性
- 學會計算和應用移動平均線 (Moving Average)
- 學會計算和應用布林通道 (Bollinger Bands)

### 內容

1. **移動平均線 (Moving Average, MA)**:
    - 什麼是 MA？為什麼它能判斷趨勢？
    - 黃金交叉 (Golden Cross) vs. 死亡交叉 (Death Cross)
    - 使用 Pandas 的 `rolling()` 方法計算 MA
2. **布林通道 (Bollinger Bands)**:
    - 由中軌 (MA)、上軌、下軌組成的通道。
    - 通道擴張 vs. 收縮 代表的意義。
    - 價格觸及上/下軌的交易訊號。
