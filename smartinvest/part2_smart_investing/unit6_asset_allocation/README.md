# 單元 6: 資產配置

「不要把所有雞蛋放在同一個籃子裡。」資產配置是投資中唯一免費的午餐，是構建穩健投資組合的基石。

## 學習目標

- 理解資產配置的核心概念與重要性
- 了解風險與報酬的關係
- 實作一個簡單的靜態資產配置模型

### 內容

1. **什麼是資產配置 (Asset Allocation)?**:
    - 在不同類型的資產之間分配你的投資資金。
    - 資產類別：股票、債券、現金、房地產...
    - 相關性 (Correlation) 的概念。
2. **風險與報酬**:
    - 高風險、高報酬；低風險、低報酬。
    - 如何衡量投資組合的風險 (標準差)？
3. **（專案）靜態資產配置實作**:
    - 建立一個包含不同資產（例如：股票ETF、債券ETF）的投資組合。
    - 計算投資組合的總報酬率。
